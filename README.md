# Orquia Portfolio

#### <u>Single-Page App:</u>

    Vite, React, TypeScript, SCSS, BootStrap and React-BootStrap.

# Development Goals:

Create and deploy a single page app within a week with React.

## App Commands

|             | Client          |
| ----------- | --------------- |
| Development | `npm run dev`   |
| Build       | `npm run build` |

## Live App

You can try the app live [here](https://orquia.netlify.app/).
